# Ad Format Selection Guide

## Quick Decision Matrix

| Funnel Stage | Primary Goal | Recommended Format |
|--------------|--------------|-------------------|
| TOFU | Attention + Education | Video, Carousel, Document |
| MOFU | Trust + Qualification | Carousel, Video, Document |
| BOFU | Conversion | Single Image, Video |

---

## Format Details

### Single Image

**Best for:**
- Clear, single promise
- One CTA
- Conversion pushes (BOFU)
- Fast comprehension

**Use when:**
- Message is simple and direct
- You want quick understanding
- Running retargeting to warm audiences
- Offer is straightforward (demo, trial)

**Specs:**
- 1200 x 1200px (1:1) or 1200 x 627px (1.91:1)
- 1:1 performs better on mobile

**Not ideal for:**
- Complex value propositions
- Educational content
- Cold audiences needing persuasion

---

### Carousel

**Best for:**
- Teaching and storytelling
- Multi-step persuasion
- Case study breakdowns
- Framework explanations

**Use when:**
- You need 3-5 steps to persuade
- Showing a journey: problem → insight → solution → proof → CTA
- Educational TOFU content
- Case studies with multiple proof points

**Specs:**
- 1080 x 1080px per card (1:1)
- 3-5 cards optimal (2 min, 10 max)

**Structure:**
1. Hook (problem or question)
2-4. Insight/proof/education
5. CTA

**Not ideal for:**
- Simple offers
- BOFU direct response
- Messages that work in one frame

---

### Video

**Best for:**
- Awareness and warm nurture
- Showing movement (workflows, automation, before/after)
- Demo snippets
- Founder/personal connection

**Use when:**
- You can show something dynamic
- The "aha moment" needs motion
- Building brand awareness
- Explaining complex concepts simply

**Specs:**
- 1:1 or 4:5 aspect ratio (mobile-optimized)
- 15-30s for TOFU, 30-90s for MOFU
- Captions required (80% watch muted)
- First 3 seconds must hook

**Types by stage:**
- **TOFU:** Short (15-30s) scroll-stopping hooks
- **MOFU:** Demo snippet or use case (30-90s)
- **BOFU:** Founder invite or quick walkthrough (30s)

**Not ideal for:**
- Quick conversion messages
- When you don't have video production capacity
- Simple announcements

---

### Document Ads

**Best for:**
- Lead generation with value preview
- Reports, checklists, mini-guides
- Gated content with preview

**Use when:**
- You have a real asset to offer
- Want to show value before the gate
- TOFU content marketing
- Lead magnet campaigns

**Specs:**
- PDF format
- First 1-3 pages visible as preview
- Make preview pages compelling standalone

**Not ideal for:**
- BOFU conversion
- Simple CTAs
- When you don't have quality content assets

---

### Message/Conversation Ads

**Best for:**
- High-intent retargeting
- Personalized outreach
- Pricing page visitors, webinar attendees

**Use when:**
- Audience is highly qualified
- Personal touch adds value
- Following up on specific actions

**Caution:**
- Can feel intrusive
- Use sparingly
- Only for warm/hot audiences
- Requires good segmentation

**Not ideal for:**
- Cold audiences
- Brand awareness
- High-volume campaigns

## Format Selection by Offer Type

| Offer Type | Primary Format | Alternative |
|------------|----------------|-------------|
| Report/Guide | Document | Carousel |
| Checklist | Document | Single Image |
| Case Study | Carousel | Document |
| Webinar | Single Image | Video |
| Demo | Single Image | Video |
| Free Trial | Single Image | Video |
| Assessment | Single Image | - |
| Comparison | Carousel | Single Image |
| POV Content | Video | Carousel |

---

## Format Selection Checklist

When choosing a format, verify:

- [ ] Format matches the message complexity
- [ ] Format fits the funnel stage
- [ ] You have the assets to execute well
- [ ] Mobile experience will be good
- [ ] CTA and offer align with format strengths

---

## When to Test Multiple Formats

Consider testing 2 formats when:
- Budget allows for meaningful split
- Message could work both ways
- You're unsure which will resonate
- Launching a new campaign type

Start with the recommended format, add alternatives only with sufficient budget.
